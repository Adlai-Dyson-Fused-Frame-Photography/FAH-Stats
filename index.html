What is Folding@home?

Folding@home is a distributed computing project that allows anyone to contribute spare CPU and GPU power to scientific research. The project focuses on simulating protein folding to better understand diseases such as cancer, Alzheimer’s, Parkinson’s, and many others.

Each calculation is processed locally on volunteers' devices and securely returned to researchers for analysis.
